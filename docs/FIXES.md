# Fixes

Beschreibt Fix-Kategorien, Sicherheitsregeln und Rueckfragen.

- AUTO-SAFE: nur klar pruefbare Aenderungen mit Backup, Test und Rollback
- GUIDED: erklaert den Schritt und fuehrt nur kontrollierte Teilaktionen aus
- MANUAL: nur Hinweis und Empfehlung
- Quarantaene: Dateien werden nie geloescht, sondern nur in einen Zeitstempelordner verschoben
