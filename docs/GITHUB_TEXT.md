# GitHub Text

## Kurzbeschreibung

Riveria Server Audit & Hardening Tool ist ein modulares Bash-Tool fuer Ubuntu- und Debian-Server. Es erkennt typische Serverrollen und Komponenten, fuehrt read-only Sicherheitschecks aus, erzeugt Reports in TXT/JSON/HTML und bietet gefuehrte Hardening-Fixes mit Backup, Test und Rollback.

## Repository Description

Professional Bash-based server audit and hardening tool for Ubuntu/Debian with auto-detection, security checks, guided fixes, and TXT/JSON/HTML reports.

## Release Post Kurztext

Riveria Server Audit & Hardening Tool `1.0.0-alpha` ist als frueher Alpha-Stand verfuegbar. Der Fokus liegt auf sicherem Bash-basiertem Audit, profilabhaengiger Erkennung, verstaendlichen Findings und gefuehrten Fix-Flows mit Backup, Test und Rollback.

Enthalten sind bereits:

- modulare Projektstruktur
- System-, Service-, App-, Code-, Exposure-, Docker-, Mail- und SSL/DNS-Checks
- gefuehrter Fix-Assistent
- mehrere echte AUTO-SAFE-Hardening-Flows
- Reports in TXT, JSON und HTML
- lokaler Smoke-Test
