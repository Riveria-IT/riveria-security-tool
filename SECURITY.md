# Security Policy

## Sicherheitsprobleme melden

Bitte melde vermutete Sicherheitsprobleme nicht oeffentlich als GitHub-Issue.

Nutze stattdessen diese Kontaktadresse:

- `m.berger.game@gmail.com`

Bitte sende in der Meldung, soweit moeglich:

- eine kurze Beschreibung des Problems
- betroffene Version oder Commit-Stand
- Schritte zur Reproduktion
- moegliche Auswirkung
- vorhandene Logs oder Screenshots ohne Secrets

## Erwarteter Umgang

- Sicherheitsmeldungen werden vertraulich behandelt.
- Bitte gib angemessen Zeit fuer Pruefung und Behebung, bevor Details oeffentlich gemacht werden.
- Secrets, Zugangsdaten, private Keys oder produktive Kundendaten sollten nicht mitgesendet werden.
